# E-Okul App Demo Frontend VueJS

## `npm install`
diyerek bağımlılıkların yüklenmesi

## `npm run serve`
diyerek projenin ayağa kaldırılması

- Layouts      (Sayfa Şablonları )

- Plugins 
- Providers     
- Router 
- Store (Vuex Modules)
- Views
